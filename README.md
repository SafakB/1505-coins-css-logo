# Using

### Upload and Add Cs  
css/coins.min.css or css/coins.css
```html
<link rel="stylesheet" href="css/coins.min.css">
```

### Upload Images  
coins16.png  
coins32.png  
coins48.png  


### For 16px
```html
<span class="ci-16 ci-16-eth"></span>    
```
### For 32px
```html
<span class="ci-32 ci-32-eth"></span>    
```
### For 48px
```html
<span class="ci-48 ci-48-eth"></span>    
```

### For Empty Icon
> A question mark will appear in the icon area. 
```html
<span class="ci-16 ci-16-0"></span>    
```
or
```html
<span class="ci-16"></span>    
```
